What did you do last week?

I was able to learn more about interacting with routers through the command line (Cisco IOS) and the different kinds of access that is granted, i.e User EXEC, Privileged EXEC mode, Global Configuration Mode, etc.. With the help of Jeremy's IT Lab's CCNA LAN Switching course, I was able to learn more about ARP (Address Resolution Protocol) and how it stitches together a server's IP address with the server MAC Address (unknown at initial request) from the perspective of the requesting client. I unfortunately was not able to get much more done on my private enterprise network as I've had to focus on other class exams and interviews.

Reflection on the process you used last week, how can you make the process work better?

This means working and learning the CCNA course and labs out of order and filling out the missing pieces ad-hoc through my own research. Hopefully this works but I'm open to other ideas.
